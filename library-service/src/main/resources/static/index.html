📚 Library Student & Book Service

Welcome to the Library Service. This service manages students who can borrow books from various libraries.

Available APIs:

Get Student by ID

GET /students/{id}

Retrieve a student by their ID

Find Students

GET /students?page=<page>&size=<size>

Find students with optional filtering

Find by Library

GET /students?library=<library>&page=<page>&size=<size>

Find students at a specific library

Create Student

POST /students

Create a new student record
